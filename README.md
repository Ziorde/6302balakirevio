# 6302balakirevio
## Лабораторные работы по курсу Технологии программирования на Python 2026

### Основные модули:

**main.py** - точка входа в программу: 

- Загружает изображение картины

- Запускает все методы обработки

- Сравнивает реализацию OpenCV с собственной

**config.py** - настройки проекта

- Пути к директориям

- Параметры обработки (размеры ядер, коэффициенты)

- Ядра Собеля для выделения границ

### Обработка изображений:

**image_processing.py** - реализация на OpenCV

- Преобразование в оттенки серого

- Гауссово размытие

- Выделение границ (Canny)

- Гамма-коррекция

- Выравнивание гистограммы

**custom_image_processing.py** - собственная реализация

- Те же функции, реализованные в ручную

### Вспомогательные модули:

**utils.py** - утилиты

- Загрузка и сохранение изображений

- Замер времени выполнения функций

**met_api.py** - работа с API музея

- Получение случайной картины из коллекции

- Скачивание изображения и метаданных

### Директории:

**paintings** - скаченные изображения

**processed** - обработанные изображения собственной и библиотечной обработками
